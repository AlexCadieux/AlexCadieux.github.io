Portfolio

  • Dialogue-Scheduly

    This responsive React app was created for Dialogue as a final project at DecodeMTL. The objectives of this application was to manage appointments for the clients of Dialogue. It includes an Express/nodeJS API and a MySQL database. We also integrated Auth0 API to handle the authentification and WhenIWork API to manage the shifts of the differente health professionals working at Dialogue. Take a look at the screencast to see what the app looks like or go see the details on Github!

  • Dashboardly

    A homepage for your grandparents's favorite browser! Dashboardly is a React App that allows different users to create boards to bookmark their favorites websites. This project also includes a back-end API built with Express/nodeJS and MySQL database. The front-end is all custom CSS. This project lives on Heroku (back-end) and Netlify (front-end). You can click on the image to visit the website and create your first board. You can also go on Github: Back-end | Front-end
